Track 2 Agent 闭环

可靠的 Agent，
关键是闭环完成任务。

不要把一次回答当终点，把规划、工具、记忆与验证都放进 Workflow。

Build loop
01

定义任务

用户目标、私有上下文、权限和完成条件

02

检索与规划

读取本地知识与记忆，决定下一步行动

03

在 Radeon 上推理

调用 ROCm 本地模型，同时记录速度与显存

04

行动并验证

调用工具，检查结果、来源与约束是否满足

05

交付或恢复

完成就交付；否则询问用户、修正计划或安全退出

Demo 最好跑完一项真实任务，也展示一次可控恢复。
