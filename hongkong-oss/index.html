分享者：Open Source Hong Kong 研究团队

说明本次分享基于 Open Source Hong Kong 的系统性研究，目标是全面呈现香港开源生态。 强调主线：演进历史、参与者、产业采用、国际连接与挑战。 提醒听众：最终会给出为什么开源重要，以及香港可如何发挥更大作用的建议。

香港开源生态的三段式演进

从兴趣驱动到组织化协作
用三阶段描述香港开源生态的演进：从社区自发到企业投入，再到治理与协作。 强调这是“从兴趣到制度化”的过程，也为后续挑战埋下伏笔。

行业采用：金融科技

开放银行与 API 标准

沙盒、第三方接入、生态合作

风控与合规自动化

KYC/AML、可审计流程

安全与隐私计算

数据脱敏、联邦学习、可信执行

开源降低了研发成本，也提升了合规与透明度
金融科技是香港开源采用最明显的行业之一，强调开放标准与合规自动化。 用案例思路说明开源降低了研发成本并提升可审计性。

行业采用：智慧城市

城市数据平台

开放数据、共享标准

交通与公共服务

可复用中间件、统一接口

数字治理

透明度、可协同、可审计

跨机构协作依赖可复用的开源基础设施
智慧城市强调跨机构协作，开源可以帮助政府与产业建立可复用的数字底座。 提醒听众关注开放数据与可治理基础设施的重要性。

行业采用：人工智能

开源模型与工具链

基础模型、推理框架、评测基准

产学研协作

联合实验室、开源数据集

MLOps 与可靠性

可追踪、可复现、可治理

开放协作缩短从研究到产业的距离
人工智能相关开源项目多与高校与企业合作相关。 说明香港在科研与产业结合方面具备桥接优势。

挑战与摩擦

关键问题：如何把兴趣驱动升级为长期机制
系统梳理挑战：可持续性、企业参与度、政策与资源、人才供给、治理合规。 提醒这些挑战会决定生态能否从活跃走向可持续与可治理。

观察与建议

  • 开源是香港创新与数字治理的公共基础设施
  • 建立 OSPO 与社区基金，提升企业与初创参与
  • 开放数据与标准化接口，形成跨机构协作
  • 与全球基金会/区域社区建立双向贡献通道
  • 将开源教育纳入高校课程与职业培养
为什么开源重要：让协作可持续，让价值可扩散
给出可行动的建议：强调开源的重要性与香港的桥梁角色。 鼓励建立制度化机制（如 OSPO、基金、政策支持）。
