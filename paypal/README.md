### Part 2: 神话破解与基础理论
- **迷思一：LLM 会幻觉？**
  - 误解：金融容不下诗人的想象
  - 真相：Agentic Workflow = 诗人穿拘束衣
  - 学术支撑：MADRA 多智能体辩论
- **迷思二：LLM 太慢？**
  - 误解：风控需要<100ms，LLM 要几秒
  - 真相：分层防御架构（L1 极速 / L2 智能 / 事后阻断）
- **简单线性归因** - "看谁分高"
- **朴素贝叶斯** - "玄学算命"（概率论）
- **实战对决：线性 vs 贝叶斯** 🥊
  - 案例分析：可疑交易判定
  - 最佳实践：两者结合
- **模型的"阿喀琉斯之踵"** 🎯
  - 线性模型的坑：特征独立假设、权重难调、缺失值处理
  - 贝叶斯的坑：朴素假设、冷启动、概率估计敏感
  - Dify 混合策略：L1 线性 + L2 贝叶斯 + L3 LLM
- **特征工程：风控的"炼金术"** 🧪
  - 好特征的三个标准：区分度高、难以伪造、稳定可获取
  - 烂特征的三个陷阱
  - PayPal 场景下的"黄金特征库"
